============
Installation
============


Stable release
--------------

To install threedi_results, run this command in your terminal:

.. code-block:: console

    $ pip install threedi_results

This is the preferred method to install threedi_results, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for threedi_results can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/larsclaussen/threedi_results

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/larsclaussen/threedi_results/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/larsclaussen/threedi_results
.. _tarball: https://github.com/larsclaussen/threedi_results/tarball/master
